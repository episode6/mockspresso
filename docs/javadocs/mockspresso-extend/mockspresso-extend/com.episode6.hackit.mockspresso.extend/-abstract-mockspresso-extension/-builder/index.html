DEPRECATED - kotlin usage is easier to maintain and should be preferred. This module will be removed in a future release.Extend this abstract class for a custom implementation of the Mockspresso.Builder interface.In this class you will only need to override the constructor, providing lambdas that wrap Mockspresso with your custom subclass of AbstractMockspressoExtension and Mockspresso.Rule with your custom subclass of AbstractMockspressoExtension.Rule.Of course you'll also need to override any custom methods you've added to your api.

Parameters

<EXT>

Should point to your custom extension of MockspressoExtension

<RULE>

Should point to your custom extension of MockspressoExtension.Rule

<BLDR>

Should point to your custom extension of MockspressoExtension.Builder
