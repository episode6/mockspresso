DEPRECATED - kotlin usage is easier to maintain and should be preferred. This module will be removed in a future release. Extend these 3 interfaces to build your own Mockspresso extension and own your api. MockspressoExtensionMockspressoExtension.RuleMockspressoExtension.Builder Each interface/class is generically typed which is what allows you to extend them whithout needing to override all the default methods. Note: if you're working in a 100% kotlin codebase, this is totally unnecessary and you're better off simply adding extension methods to Mockspresso.Builder. These interfaces/abstract classes are only needed for java. Usually you will only want to add custom methods to your extension of MockspressoExtension.Builder
